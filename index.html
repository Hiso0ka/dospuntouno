Navegadores WEB

Un navegador web, es un software, que permite el acceso a la web, interpretando la información de distintos tipos de archivos.

La funcionalidad básica de un navegador web es permitir la visualización de documentos de texto, posiblemente con recursos multimedia incrustados.

Ejemplos de navegadores WEB

Safari:Safari es un navegador de código cerrado desarrollado por Apple Inc. Está disponible para macOS,IpadOS e iOS

Mozilla firefox es un navegador de web libre y de código abierto desarrollado para Linux, Android, etc.

Google Chrome:Navegador de código cerrado desarrollado por Google, aunque derivado de proyectos de código abierto.
